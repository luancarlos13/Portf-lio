# 💼 Portfólio Pessoal

## 📌 Sobre o Projeto

Este é o meu portfólio pessoal desenvolvido para apresentar minha trajetória como desenvolvedor front-end, minhas habilidades técnicas e os projetos que venho criando ao longo da minha jornada na programação.

O projeto foi construído com foco em design moderno, animações suaves e uma experiência de navegação intuitiva, permitindo que visitantes conheçam meu trabalho de forma dinâmica e profissional. Além de servir como vitrine para meus projetos, o portfólio também representa minha paixão por tecnologia, inovação e desenvolvimento de soluções criativas para a web.

## ✨ Destaques

- 👨‍💻 Apresentação profissional e personalizada.
- 🚀 Exibição dos principais projetos desenvolvidos.
- 🎨 Interface moderna e visual atrativo.
- 📱 Layout totalmente responsivo.
- ⚡ Navegação fluida e intuitiva.
- 🌐 Links para redes profissionais e contato.
- 💡 Demonstração das minhas habilidades em desenvolvimento front-end.

## 🛠️ Tecnologias Utilizadas

- HTML5 🟧
- CSS3 🟦
- JavaScript 🟨
- Git 🔮
- GitHub Pages 📌

## 🎯 Objetivo

Criar um espaço digital que reúna meus projetos, experiências e conhecimentos, demonstrando minha evolução como desenvolvedor e meu compromisso com a criação de experiências web modernas, responsivas e inovadoras.
